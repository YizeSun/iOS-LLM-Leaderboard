# Pilot v0.1 Raw App Results

Place only unmodified, real `suite-b-result-bundle-0.2` or
`suite-b-result-bundle-0.3` JSON exports from the Benchmark App in this
directory. The internal Pilot accepts only B-UX-001 and B-PIPE-001.

This directory contains the final six-file App 0.6.0 build 8 matrix from an
iPhone 14 Pro Max (`iPhone15,3`): three fixed Qwen3 artifacts across B-UX-001
and B-PIPE-001. All files are genuine App exports and remain byte-identical to
the supplied originals. The earlier two-file App 0.5 smoke run is retained in
`../stage-1-smoke/` and is not ingested into the final comparison. Do not add
placeholders, demonstrations, simulated measurements, hand-edited exports, or
community-submission wrappers.

Follow the concise
[Stage 1 physical-iPhone runbook](../../../docs/power-ship-pilot-v0.1.md#stage-1-physical-iphone-runbook).
Export **Raw JSON** from the App for B-UX-001 and B-PIPE-001; the Legacy Draft
Submission JSON is not accepted by the Pilot ingestion path.
