# App Store Review Response

## Scenario

An iOS developer wants help drafting a response to a critical App Store review.

## Prompt

Draft a concise, professional response to a one-star App Store review complaining that a subscription feature did not unlock after purchase.

## Expected Behavior

- Acknowledges the issue respectfully.
- Avoids blaming the user.
- Suggests practical support steps.
- Does not request sensitive information publicly.

## Scoring Rubric

Evaluate usefulness, tone, policy awareness, and practical next steps.

## Notes for Reviewers

Responses should be appropriate for a public App Store review thread.
