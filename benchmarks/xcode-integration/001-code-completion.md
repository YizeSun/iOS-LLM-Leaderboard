# Code Completion

## Scenario

An iOS developer evaluates model-assisted code completion inside an Xcode workflow.

## Prompt

Complete a partially written SwiftUI view that displays model data and handles loading, error, and empty states.

## Expected Behavior

- Completes code in the local style.
- Avoids large unrelated rewrites.
- Produces compilable Swift.
- Handles common UI states.

## Scoring Rubric

Evaluate completion relevance, correctness, minimality, and integration quality.

## Notes for Reviewers

Record the editor integration, model, and completion settings.
