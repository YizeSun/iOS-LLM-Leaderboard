# SwiftData CRUD

## Scenario

An iOS developer wants a simple SwiftData-backed CRUD flow.

## Prompt

Create a minimal SwiftUI and SwiftData example that can create, display, update, and delete notes.

## Expected Behavior

- Defines a SwiftData model.
- Uses `@Query` or appropriate SwiftData APIs.
- Supports create, read, update, and delete operations.
- Keeps the architecture understandable.
