# SwiftUI List Delete

## Scenario

An iOS developer needs a SwiftUI list where users can delete rows with standard system interactions.

## Prompt

Create a SwiftUI view that displays a list of editable items and supports deleting rows using the native swipe-to-delete interaction.

## Expected Behavior

- Uses SwiftUI `List`.
- Supports deletion with `onDelete`.
- Updates state correctly after deletion.
- Keeps the example minimal and compilable.
