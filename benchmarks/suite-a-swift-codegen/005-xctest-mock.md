# XCTest Mock

## Scenario

An iOS developer needs unit tests for a service that depends on a network client.

## Prompt

Create a small Swift example with a protocol-based network client and XCTest tests using a mock implementation.

## Expected Behavior

- Uses dependency injection.
- Provides a mock or fake implementation.
- Includes meaningful XCTest assertions.
- Avoids real network calls in tests.

## Scoring Rubric

Use the Suite A Swift Code Generation Score in `methodology/scoring.md`.

## Notes for Reviewers

Assess whether the test would catch realistic regressions.
