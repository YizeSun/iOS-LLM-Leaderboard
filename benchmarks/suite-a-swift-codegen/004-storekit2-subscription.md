# StoreKit 2 Subscription

## Scenario

An iOS app needs a basic StoreKit 2 subscription purchase flow.

## Prompt

Write Swift code for loading subscription products, starting a purchase, and checking entitlement status using StoreKit 2.

## Expected Behavior

- Uses StoreKit 2 APIs.
- Handles purchase states.
- Verifies transactions.
- Avoids storing entitlement state without validation.

## Scoring Rubric

Use the Suite A Swift Code Generation Score in `methodology/scoring.md`.

## Notes for Reviewers

StoreKit code should be reviewed carefully for incorrect entitlement assumptions.
