# MLX Swift Integration Recipes

This directory is reserved for focused, copy-ready MLX Swift examples.

Planned recipes:

- add the package dependency;
- load a tested model profile;
- perform one-shot generation;
- stream tokens;
- cancel generation;
- reset context and cache state;
- download or bundle model artifacts;
- handle memory and thermal conditions;
- export benchmark-compatible metadata where useful.

Examples should remain small and composable. They should not require developers
to adopt a complete project architecture.

No implementation is included yet.
