# Core ML Integration Recipes

This directory is reserved for focused Core ML examples backed by documented
model conversions and device requirements.

Planned recipes:

- load a compatible model;
- configure compute units;
- prepare input;
- run inference;
- handle model resources and errors;
- record the conversion and model-profile identity;
- export benchmark-compatible metadata where applicable.

Examples should explain conversion-specific limitations and should remain
small enough to adapt to an existing app.

No implementation is included yet.
